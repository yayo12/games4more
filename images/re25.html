Requisitos y Descripcion

Descripcion

Far Cry 5 es un videojuego de acción-aventura en primera persona desarrollado por Ubisoft Montreal y publicado por Ubisoft para PlayStation 4, Xbox One y Microsoft Windows. Es la undécima entrega de la serie Far Cry. Su lanzamiento se produjo el 27 de marzo de 2018.1​

Requisitos Minimos

Sistema operativo: Windows 7 SP1, Windows 8.1, Windows 10 (64-bit)

Procesador: Intel Core i5-2400 @ 3.1 GHz o AMD FX-6300 @ 3.5 GHz o equivalente

Memoria: 8GB RAM

Tarjeta gráfica: NVIDIA GeForce GTX 670 o AMD R9 270 (2GB VRAM con Shader Model 5.0 o superior)

Resolución: 720p / Calidad de vídeo: baja

Requisitos Recomendado

Sistema operativo: Windows 7 SP1, Windows 8.1, Windows 10 (64-bit versions solo)

Procesador: Intel Core i7-4770 @ 3.4 GHz o AMD Ryzen 5 1600 @ 3.2 GHz o equivalente

Memoria: 8GB RAM

Tarjeta gráfica: NVIDIA GeForce GTX 970 o AMD R9 290X (4GB VRAM con Shader Model 5.0 o superior)

Resolución: 1080p / Calidad de vídeo: baja
